Bascula IoT

Este proyecto esta pensado para conectar una bascula que tenga como minimo un puerto de comunicación, para este proyecto se utiliza un puerto USB.
La Bascula se encuentra conectada internamente a una Raspberry la cual es la que tine cargada lo que es el servidor local para simular una pagina en la que se muestran
todas las ventas que se han hecho con la misma bascula.
